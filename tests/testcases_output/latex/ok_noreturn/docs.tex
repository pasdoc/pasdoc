\label{ok_noreturn}
\index{ok{\_}noreturn}
\section{Description}
Test that the noreturn directive is accepted on procedures.\hfill\vspace*{1ex}

 Delphi uses the noreturn directive to indicate that a procedure never returns to the caller. See \href{https://docwiki.embarcadero.com/RADStudio/Florence/en/Declarations_and_Statements_(Delphi)#The_noreturn_Directive}{Delphi docs}.
